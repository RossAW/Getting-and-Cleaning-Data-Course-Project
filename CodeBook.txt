##Getting and Cleaning Data Course Project Code Book
This script merges the training and the test data sets to create one data set.
- selects the variables of interest i.e. Subject, Activity and the mean and standard deviation of all the variables
- makes the data easier to read
- creates a second, independent tidy data set with the average of each variable for each activity and each subject.


The TidyOutputData file contains the averages for each of the following variables for each activity and
